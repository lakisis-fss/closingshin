# 대시보드 지수 텍스트 크기 확대 보고서

## 1. 개요
사용자 편의성과 데이터 시인성을 높이기 위해 우상단 지수 표시 영역의 폰트 크기를 상향 조정하였습니다.

## 2. 작업 상세
### 2.1 대상 파일
- `frontend/src/components/dashboard/MarketStatusBar.tsx`

### 2.2 스타일 변경 내용
- **Font Size Scaling**: `text-xs`(12px) → `text-[15px]`
  - 물리적으로 약 3포인트 증가시켜 가독성 확보
  - `font-black` 속성과 결합하여 고대비 시인성 극대화
- **Layout Consistency**: 
  - 폰트 크기가 커짐에 따라 인접한 요소들과의 간격을 재조정할 필요가 없도록 `relative sizing` 원칙 준수

## 3. 디자인 분석 및 기대 효과
| 구분 | 이전 (12px) | 현재 (15px) | 효과 |
| :--- | :--- | :--- | :--- |
| **시인성** | 정독 필요함 | **한눈에 인식 가능** | 정보 습득 속도 향상 |
| **디자인 밸런스** | 로고 대비 왜소함 | **로고와 적절한 균형** | 상단 바의 전체적 안정감 확보 |
| **사용자 경험** | 모니터 글씨가 작음 | **시원한 가독성** | 장시간 모니터링 시 피로도 감소 |

## 4. 최종 결과 요약
이번 텍스트 확대 작업을 통해 "STAY ANALYTICAL."이라는 브랜드 로고와 나란히 놓인 지수 데이터들이 시각적으로 더 조화로운 비율을 갖게 되었습니다. 이제 어떤 각도에서도 시장 지표를 선명하게 확인할 수 있습니다.
