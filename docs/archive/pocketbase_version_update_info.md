# PocketBase 최신 버전(0.23) 정보

이 문서는 PocketBase 0.23 버전 출시 소식과 주요 특징, 그리고 현재 프로젝트 버전(0.22.10)에서의 업데이트 시 고려사항을 정리한 것입니다.

## 1. 버전 비교 요약

| 구분 | 현재 프로젝트 버전 | 최신 버전 (0.23) |
| :--- | :--- | :--- |
| **버전 명칭** | `0.22.10` | `0.23.0` (정식 출시) |
| **주요 상태** | 안정적 운영 중 | 대규모 기능 추가 및 최적화 |

---

## 2. 0.23 버전의 주요 특징 (무엇이 바뀌었나요?)

PocketBase 0.23은 엔진이 통째로 리팩토링된 수준의 업데이트입니다.

- **강력한 보안**: OTP(이메일 등을 통한 일회용 번호) 및 MFA(2단계 인증) 지원
- **효율성 증대**: Batch Web API를 통해 여러 요청을 한 번에 처리하여 네트워크 부하 감소
- **속도 제한**: 내장된 Rate Limiter를 통해 서버를 스팸이나 과도한 요청으로부터 보호
- **새로운 API**: Go 및 JavaScript/TypeScript 전용 API가 더 사용하기 편하게 개선됨

---

## 3. 업데이트 시 주의사항 (중요!)

현재 `0.22.10`에서 `0.23`으로 바로 업데이트할 경우 **수동 작업**이 필요합니다.

> [!WARNING]
> **브레이킹 체인지 (Breaking Changes)**
> - `pb_hooks` 및 `pb_migrations` 내부의 코드 형식이 바뀌었습니다.
> - 따라서 실행 파일(`pocketbase.exe`)만 바꾼다고 해서 기존 데이터나 코드가 100% 돌아가지 않을 수 있습니다.
> - 반드시 공식 가이드에 따라 코드를 수정하는 '마이그레이션' 과정을 거쳐야 합니다.

---

## 4. 권장 사항

1. 현재 프로젝트가 잘 돌아가고 있다면 급하게 업데이트를 서두르지 마세요.
2. 신규 기능(MFA, OTP 등)이 꼭 필요하다면 별도의 테스트 환경에서 먼저 업데이트를 시도해 보세요.
3. 업데이트 전에는 반드시 `pb_data` 폴더를 백업해 두어야 합니다.
