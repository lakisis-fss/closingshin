# 포트폴리오 가격 표시 로직 수정 보고서

## 개요
포트폴리오 페이지에서 보유 종목의 현재가를 표시할 때 시장 상황(장중 또는 장마감 후)에 따라 가장 적절한 값을 선택하도록 로직을 개선했습니다.

## 배경
기존 시스템은 장이 열려 있을 때와 닫혀 있을 때를 구분하지 않고 데이터를 가져왔습니다. 이로 인해 장이 마감된 후에도 실시간 데이터가 불규칙하게 표시되거나 공식적인 종가와 다른 값이 보일 수 있는 문제가 있었습니다. 이를 해결하기 위해 현재 시간을 기준으로 장 운영 여부를 판단하는 기능을 추가했습니다.

## 주요 변경 사항

### 1. 시장 운영 시간 판단 기능 추가
한국 주식 시장(KOSPI, KOSDAQ)의 운영 시간을 판단하는 함수를 만들었습니다.
- 운영 시간: 평일 오전 9시 00분부터 오후 3시 30분까지
- 제외 기간: 토요일, 일요일 및 공휴일 (2026년 공휴일 데이터 반영)

### 2. 가격 선택 우선순위 정립
종목의 현재 가격을 결정할 때 다음과 같은 규칙을 적용합니다.

**장이 열려 있을 때 (장중)**
1. 실시간 시세 데이터(statusData)를 가장 먼저 확인합니다.
2. 실시간 데이터가 없을 경우에만 데이터베이스에 저장된 종가를 확인합니다.

**장이 닫혀 있을 때 (장마감 후 / 주말 / 공휴일)**
1. 데이터베이스에 저장된 공식 종가(vcp.close)를 가장 먼저 사용합니다.
2. 이는 장이 마감된 후에는 실시간 가격보다 공식적인 하루의 마무리 가격(종가)을 보는 것이 투자 판단에 더 정확하기 때문입니다.

### 3. 적용 범위
위 로직은 포트폴리오 페이지의 모든 곳에 일관되게 적용되었습니다.
- 포트폴리오 상단 요약 대시보드 (총 자산, 총 수익률 등)
- 보유 종목 리스트 테이블 (각 종목별 수익률, 현재가)
- 데이터 내보내기 기능 (CSV, XLSX 엑셀 파일)
- 종목 청산 모달 (청산 가격 기본값 설정)

## 코드 변경 위치
- frontend/src/lib/utils.ts: 시장 개장 여부 판단 로직 추가
- frontend/src/components/portfolio/HoldingsTable.tsx: 리스트 및 지표 계산 로직 수정
- frontend/src/components/portfolio/PortfolioDashboard.tsx: 요약 통계 로직 수정
- frontend/src/app/portfolio/PortfolioPageClient.tsx: 엑셀 내보내기 로직 수정

## 기대 효과
- 사용자는 시장 운영 시간에는 실시간 변동을 즉각적으로 확인할 수 있습니다.
- 장마감 후에는 확실한 종가 데이터를 바탕으로 하루의 투자 성과를 안정적으로 검토할 수 있습니다.
- 모든 화면에서 동일한 가격을 보여줌으로써 데이터의 신뢰성을 높였습니다.
