# VCP 대시보드 Sector/RS 점수 처리 방식 분석

## 1. 현재 처리 방식 (Status Quo)
현재 대시보드의 **Sector/RS (상대 강도)** 점수는 프론트엔드 코드상에서 **'50점(중립)'으로 고정**되어 있습니다. 

## 2. 왜 50점으로 고정되어 있나요?
고등학생 수준에서 예를 들어 설명해 보겠습니다.

> **축구 선수의 능력치 평가 예시**
> 한 선수의 '달리기 속도(VCP)', '체력(수급)', '인기도(AI 심리)'는 이미 100점 만점으로 계산하는 기준이 정해져 있습니다. 하지만 '팀 내 기여도(Sector/RS)'는 실력 데이터는 수집되고 있지만, 이걸 "팀 평균에 비해 몇 점인가?"라는 100점 만점 점수로 변환하는 공식이 아직 대시보드 화면에 연결되지 않은 상태입니다. 
> 그래서 시스템은 일단 이 선수에게 "평범하다(50점)"라는 점수를 주어 전체 평균 점수(Total Quant Score)를 계산할 때 방해가 되지 않게 처리하고 있는 것입니다.

## 3. 기술적 배경 (Technical Insight)
- **백엔드 데이터**: `vcp_reports` 데이터베이스에는 `relative_strength`라는 실제 계산된 값이 들어 있습니다. (미너비니의 상대 강도 공식을 기반으로 함)
- **프론트엔드 미연동**: 대시보드 화면을 구성하는 코드(`VcpDashboard.tsx` 및 `scoreCalculator.ts`)에서 이 값을 가져와서 0~100 사이의 점수로 바꾸는 '변환기(Normalization)' 로직이 아직 작성되지 않았습니다.

## 4. 향후 계획
- 현재 10%의 가중치를 가지고 있지만 점수가 50점으로 고정되어 있어 전체 점수에 미치는 영향은 미미합니다.
- 추후 시장 지수(KOSPI/KOSDAQ) 대비 해당 종목이 얼마나 강하게 상승했는지를 백분율 점수로 변환하여 연동하면, 더욱 정교한 `TOTAL QUANT SCORE`를 보실 수 있게 될 예정입니다.
